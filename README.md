## 🛠️ Tecnologias

- **Jekyll** – Static site generator
- **Minima Theme** – Layout limpo e responsivo
- **jekyll-seo-tag** – Otimização para mecanismos de busca (ATS-friendly)
- **jekyll-sitemap** – Sitemap automático para indexação
- **GitHub Pages** – Deploy automático

## 🚀 Como rodar localmente

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
# Acesse: http://localhost:4000
```
